# Whitehat Mini Product App

## How to run:

to start the server:

1. cd backend 
2. in the terminal under the backend folder run 'npm install'

frontend:

if you dont have live server extention in vs code, install it.

right click on add-products-page.html -----> start with live server


## Features

- Add products with name and price
- Search products by name
- Prevent duplicate product names
- Input validation on frontend and backend
- SQL injection protection using parameterized queries
- XSS protection by rendering user data with `textContent`
- Request body size limit
- Rate limiting
